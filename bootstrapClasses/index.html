Bootstrap: Guía de Clases

Bootstrap es un framework de diseño web que proporciona una serie de clases CSS y componentes predefinidos para ayudar a los desarrolladores a crear interfaces de usuario responsivas y con un aspecto coherente

Algunas de las principales clases que se utilizan en Bootstrap son:

Bootstrap utiliza un sistema de cuadrícula para ayudar a los desarrolladores a crear diseños de página responsivos. Las clases relacionadas con la cuadrícula pueden ser:
  1. Container class:

    La clase "container" se utiliza para contener todo el contenido de la página y para aplicar márgenes laterales automáticos que centrarán el contenido en la página.

  2. Row class:

    La clase "row" se utiliza para crear una fila de columnas. Dentro de una fila, cada columna se divide en 12 partes iguales.

  3. Column classes:

    Las clases de columna, como "col-md-6" o "col-lg-4", se utilizan para definir el ancho de una columna en función del ancho de la pantalla. El número después de "col-" define cuántas partes de las 12 totales de la fila ocupa la columna. El prefijo "md" o "lg" se refiere a los puntos de ruptura de la pantalla en los que se aplicará el ancho de la columna.

    ¡Dato de color!

    Los prefijos usados son los siguientes:

    • xs: Este prefijo se utiliza para definir el ancho de las columnas en pantallas extra pequeñas. El ancho por defecto de las columnas en este tipo de pantallas es el 100% de la anchura del contenedor padre, por lo que no se requiere el uso de esta clase en estos casos.
    • sm: Este prefijo se utiliza para definir el ancho de las columnas en pantallas pequeñas. El ancho por defecto de las columnas en este tipo de pantallas es el 100% de la anchura del contenedor padre, por lo que no se requiere el uso de esta clase en estos casos.
    • md: Este prefijo se utiliza para definir el ancho de las columnas en pantallas medianas. En Bootstrap, se consideran pantallas medianas aquellas con un ancho mínimo de 768 píxeles.
    • lg: Este prefijo se utiliza para definir el ancho de las columnas en pantallas grandes. En Bootstrap, se consideran pantallas grandes aquellas con un ancho mínimo de 992 píxeles.
    • xl: Este prefijo se utiliza para definir el ancho de las columnas en pantallas extra grandes. En Bootstrap, se consideran pantallas extra grandes aquellas con un ancho mínimo de 1200 píxeles.
  4. Offset classes:

    Las clases de desplazamiento, como "offset-md-2" o "offset-lg-3", se utilizan para agregar un margen izquierdo a una columna y crear espacios en blanco entre las columnas. El número después de "offset-" define cuántas partes de las 12 totales de la fila se desplaza la columna hacia la derecha.

  5. Nesting columns:

    Las columnas también se pueden anidar dentro de otras columnas utilizando las clases "row" y "col". Esto es útil para crear diseños más complejos.

  6. Reordering columns:

    Bootstrap también proporciona clases de reordenamiento, como "order-first" o "order-last", para cambiar el orden de las columnas en función del ancho de la pantalla.

Bootstrap proporciona una serie de clases CSS para ayudar a personalizar la apariencia del texto en una página web. Estas clases incluyen:
  1. .h1 a .h6:

    Estas utilidades de estilo se utilizan para establecer el tamaño de encabezado de un elemento. El número después de h indica el nivel de encabezado, por ejemplo, h1 es el encabezado de nivel superior, mientras que h6 es el encabezado de nivel inferior.

  2. .lead:

    Esta utilidad de estilo se utiliza para establecer el tamaño de fuente y el margen inferior de un elemento de texto para destacarlo. Se utiliza comúnmente para introducciones y resúmenes de contenido.

  3. .text-*:

    Esta utilidad de estilo se utiliza para establecer el color de texto de un elemento, como se mencionó anteriormente en las utilidades de estilo.

  4. .font-weight-bold y .font-weight-normal:

    Estas utilidades de estilo se utilizan para establecer el peso de la fuente de un elemento. .font-weight-bold establece la fuente en negrita, mientras que .font-weight-normal establece la fuente en peso normal.

  5. .text-center, .text-left y .text-right:

    Estas utilidades de estilo se utilizan para alinear el texto de un elemento. .text-center centra el texto, .text-left alinea el texto a la izquierda y .text-right alinea el texto a la derecha.

  6. .text-truncate:

    Esta utilidad de estilo se utiliza para truncar el texto de un elemento y mostrar puntos suspensivos al final del texto si el contenido excede el ancho del elemento. Se utiliza comúnmente en elementos de navegación y tarjetas para asegurarse de que el contenido se ajuste correctamente en un espacio limitado.

Bootstrap ofrece una variedad de componentes predefinidos para agregar funcionalidad a su sitio web. Algunos de los componentes a utilizar pueden ser:
  1. .navbar:

    Esta clase se utiliza para crear una barra de navegación, que suele encontrarse en la parte superior de un sitio web. Puede contener enlaces, botones, menús desplegables y otros componentes de navegación.

  2. .card:

    La clase .card se utiliza para crear tarjetas o paneles que contienen información, como imágenes, texto y botones. Las tarjetas son útiles para presentar contenido en una página web de manera organizada y atractiva.

  3. .carousel:

    Esta clase se utiliza para crear carruseles de imágenes, que permiten mostrar varias imágenes en una misma sección de la página. Los carruseles se pueden utilizar para resaltar productos, mostrar imágenes de un evento o simplemente añadir interactividad a la página.

  4. .modal:

    Esta clase se utiliza para crear ventanas emergentes o modales que muestran información adicional o requieren la interacción del usuario. Los modales pueden contener texto, imágenes, formularios y otros componentes.

  5. .dropdown:

    La clase .dropdown se utiliza para crear menús desplegables que permiten al usuario acceder a opciones adicionales. Los menús desplegables pueden contener enlaces, botones, formularios y otros componentes.

  6. .btn:

    Esta clase se utiliza para crear botones, que se pueden utilizar para la interacción del usuario, como enviar un formulario, cerrar una ventana emergente, acceder a una sección de la página, etc. Bootstrap ofrece diferentes estilos de botones como .btn-primary, .btn-secondary, .btn-danger, entre otros.

Bootstrap también incluye una variedad de utilidades de estilo para ayudar a personalizar y ajustar los estilos de los componentes. Algunas de las utilidades de estilo más comunes son:
  1. .bg-*:

    Esta utilidad de estilo se utiliza para establecer el color de fondo de un elemento. El * representa un nombre de color o un código hexadecimal de color. Por ejemplo, .bg-primary establece un fondo azul oscuro, mientras que .bg-danger establece un fondo rojo.

  2. .text-*:

    Esta utilidad de estilo se utiliza para establecer el color de texto de un elemento. El * representa un nombre de color o un código hexadecimal de color. Por ejemplo, .text-primary establece el color de texto azul oscuro, mientras que .text-danger establece el color de texto rojo.

  3. .d-*:

    Esta utilidad de estilo se utiliza para controlar la visualización de un elemento. El * representa una clase de visualización, como none para ocultar un elemento, block para mostrar un elemento como un bloque, inline para mostrar un elemento en línea, entre otros.

  4. .m-* y .p-*:

    Estas utilidades de estilo se utilizan para establecer la cantidad de margen y relleno de un elemento. El * representa un número que indica la cantidad de espacio, por ejemplo, .m-4 establece un margen de cuatro unidades y .p-2 establece un relleno de dos unidades.

  5. .border-*:

    Esta utilidad de estilo se utiliza para agregar un borde a un elemento. El * representa el tipo de borde, como solid, dashed, dotted, none, etc. Por ejemplo, .border-primary establece un borde sólido de color azul oscuro.

  6. rounded-*:

    Esta utilidad de estilo se utiliza para redondear las esquinas de un elemento. El * representa el radio de la esquina, por ejemplo, .rounded-3 establece un radio de esquina de tres unidades.
